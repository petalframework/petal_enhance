# Petal Enhance

## Install

1. Install dep:

```elixir
# mix.exs

{:petal_enhance, git: "https://github.com/petalframework/petal_enhance"},
```

2. Create a project on [petal.build](https://petal.build)


3. Copy your env vars 

```
# .envrc
export PETAL_BUILD_PROJECT=""
export PETAL_BUILD_API_TOKEN=""
```
